# CorrectScoreLab

## Objetivo

Construir un sistema rentable para Correct Score de forma progresiva.

## Filosofía

- Proyecto pequeño.
- Máximo 10 variables principales.
- Cada mejora debe demostrar que aporta valor.
- Sin IA hasta que sea necesaria.
- Todo debe ser entendible.
